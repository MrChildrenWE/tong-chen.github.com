1. Check if an executable program is 32 bit or 64 bit.
``` bash
file `which program-name`
```

2. Set the value for executable files or libs
``` bash
# For executable
export PATH=${PATH}:/yourpath
# Shared libraries
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/yourpatj
```
